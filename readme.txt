EXIT: To exit Subject Select, click on show hidden icon arrow on taskbar, right click on Subject Select icon and click exit

STARTUP: Automatically start Subject Select on Startup
1) Open Run dialog box by pressing Windows+R
2) Type shell:startup and press Enter
3) Now paste the shortcut in the startup folder
or navigate to-
C:\Users\%USERNAME%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

To remove app from startup navigate to startup folder and remove the shortcut
